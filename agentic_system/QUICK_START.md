# 빠른 시작 가이드

## 🚀 전체 시스템 시작 (Windows)

### 방법 1: 자동 시작 (권장)
```batch
start_all.bat
```
이 명령으로 API 서버와 프론트엔드 서버가 자동으로 시작됩니다.

### 방법 2: 수동 시작
**터미널 1 - API 서버:**
```batch
python start_api_server.py
```

**터미널 2 - 프론트엔드 서버:**
```batch
start_frontend.bat
```

## 🌐 브라우저 접속

서버 시작 후 브라우저에서 접속:
```
http://localhost:5173
```

## 📋 테스트 시나리오

### 시나리오 1: 상품 추천
1. 텍스트 입력: "빨간색 원피스 추천해줘"
2. "요청 전송" 클릭
3. 추천 상품 목록 확인

### 시나리오 2: 이미지 분석
1. 의류 이미지 업로드
2. "요청 전송" 클릭
3. 이미지 분석 결과 확인

### 시나리오 3: 가상 피팅
1. 의류 이미지 업로드
2. 텍스트 입력: "이 옷을 입혀줘"
3. "요청 전송" 클릭
4. 3D 생성 결과 확인 (Mock 모드)

## ✅ 확인 사항

- API 서버: http://localhost:8000/health (정상 시 `{"status":"healthy"}`)
- 프론트엔드: http://localhost:5173 (웹 페이지 로드)
- 브라우저 콘솔: 오류 메시지 확인 (F12)

## 🔧 문제 발생 시

1. **포트 충돌**: 다른 프로세스 종료
2. **npm 오류**: `cd frontend && npm install`
3. **API 연결 오류**: API 서버 실행 확인

## 📊 예상 동작

- ✅ 텍스트 요청: 2-3초 내 응답
- ✅ 이미지 요청: 5-10초 내 응답 (Mock 모드)
- ✅ 결과 표시: JSON 데이터 및 시각화
