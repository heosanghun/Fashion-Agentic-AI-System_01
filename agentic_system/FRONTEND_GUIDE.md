# 프론트엔드 사용 가이드

## 🚀 빠른 시작

### Windows
```batch
# 방법 1: 전체 시스템 시작 (권장)
start_all.bat

# 방법 2: 개별 시작
# 터미널 1: API 서버
python start_api_server.py

# 터미널 2: 프론트엔드 서버
start_frontend.bat
```

### Linux/Mac
```bash
# 방법 1: 전체 시스템 시작
# 터미널 1: API 서버
python start_api_server.py

# 터미널 2: 프론트엔드 서버
./start_frontend.sh
```

## 📋 사용 방법

### 1. 서버 시작 확인
- ✅ API 서버: http://localhost:8000 (헬스체크)
- ✅ 프론트엔드: http://localhost:5173

### 2. 브라우저에서 접속
```
http://localhost:5173
```

### 3. 테스트 방법

#### 텍스트만 입력
1. 텍스트 입력란에 입력: "빨간색 원피스 추천해줘"
2. "요청 전송" 버튼 클릭
3. 결과 확인

#### 이미지만 업로드
1. 이미지 업로드 영역에 의류 이미지 드래그 또는 선택
2. "요청 전송" 버튼 클릭
3. 결과 확인

#### 이미지 + 텍스트
1. 이미지 업로드
2. 텍스트 입력 (예: "이 옷을 입혀줘")
3. "요청 전송" 버튼 클릭
4. 결과 확인

## 🎯 주요 기능

### 입력
- ✅ 텍스트 입력 (의류 추천, 가상 피팅 요청 등)
- ✅ 이미지 업로드 (드래그 앤 드롭 지원)
- ✅ 이미지 미리보기

### 출력
- ✅ 요청 처리 상태 표시
- ✅ 상품 추천 결과 (텍스트 요청 시)
- ✅ 이미지 분석 결과 (이미지 업로드 시)
- ✅ 3D 시각화 (3D 생성 결과)
- ✅ 전체 결과 데이터 (JSON 형식)

## 🔧 문제 해결

### npm 패키지 오류
```bash
cd frontend
rm -rf node_modules package-lock.json
npm install
```

### 포트 충돌
- API 서버 (8000): 다른 프로세스 확인 후 종료
- 프론트엔드 (5173): vite.config.js에서 포트 변경

### CORS 오류
- API 서버의 CORS 설정 확인
- 프론트엔드 vite.config.js의 proxy 설정 확인

## 📊 예상 응답 시간

- 텍스트만: 2-3초
- 이미지 포함: 5-10초 (Mock 모드)
- 3D 생성: 10-30초 (Mock 모드)

## 🎨 UI 구성

1. **헤더**: 시스템 이름 및 설명
2. **입력 영역**: 텍스트 입력, 이미지 업로드
3. **상태 표시**: 처리 중 상태
4. **결과 영역**: 추천 상품, 3D 시각화, 분석 결과
5. **푸터**: 시스템 정보
