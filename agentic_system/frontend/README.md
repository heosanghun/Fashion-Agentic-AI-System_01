# Fashion Agentic AI - Frontend

## 🚀 빠른 시작

### 개발 서버 실행

```bash
npm run dev
```

개발 서버가 시작되면 자동으로 브라우저가 열립니다.

**Cursor IDE 브라우저 탭에서 접속:**
브라우저 탭의 주소창에 다음 URL을 입력하세요:
```
http://localhost:5173
```

## 📝 주요 기능

- React + Vite 기반 프론트엔드
- 이미지 업로드 및 텍스트 입력
- 3D 모델 뷰어
- 실시간 처리 상태 표시

## 🔧 설정

- **포트**: 5173
- **API 프록시**: `/api` → `http://localhost:8000`
