## 🚀 빠른 시작

### 방법 1: 자동 시작 (Windows - 권장)

```batch
# 프로젝트 루트에서 실행
start_all.bat
```

이 명령으로 **API 서버**와 **프론트엔드 서버**가 자동으로 시작됩니다.

### 방법 2: 수동 시작

**터미널 1 - API 서버:**
```batch
python start_api_server.py
```

**터미널 2 - 프론트엔드 서버:**
```batch
cd frontend
npm install  # 처음 한 번만
npm run dev
```

http://localhost:5173

## 📋 사용 방법

### 1. 텍스트만 입력 (상품 추천)
1. 텍스트 입력란에 입력: **"빨간색 원피스 추천해줘"**
2. **"요청 전송"** 버튼 클릭
3. 추천 상품 목록 확인

### 2. 이미지만 업로드 (이미지 분석)
1. **이미지 업로드** 영역에 의류 이미지 드래그 앤 드롭 또는 클릭하여 선택
2. **"요청 전송"** 버튼 클릭
3. 이미지 분석 결과 확인

### 3. 이미지 + 텍스트 (가상 피팅)
1. **이미지 업로드**: 의류 이미지 업로드
2. **텍스트 입력**: "이 옷을 입혀줘" 또는 "3D로 생성해줘"
3. **"요청 전송"** 버튼 클릭
4. 3D 생성 결과 확인 (Mock 모드)

## 🎯 주요 기능

### 입력 기능
- ✅ **텍스트 입력**: 상품 추천, 가상 피팅 요청 등
- ✅ **이미지 업로드**: 드래그 앤 드롭 지원
- ✅ **이미지 미리보기**: 업로드 전 미리 확인

### 출력 기능
- ✅ **상태 표시**: 처리 중 상태 실시간 표시
- ✅ **상품 추천 결과**: 텍스트 요청 시 추천 상품 목록
- ✅ **이미지 분석 결과**: 이미지 업로드 시 분석 결과
- ✅ **3D 시각화**: 3D 생성 결과 (Three.js 뷰어)
- ✅ **전체 결과 데이터**: JSON 형식으로 상세 결과 확인

## 🔧 문제 해결

### npm 패키지 오류
```bash
cd frontend
rm -rf node_modules package-lock.json
npm install
```

### 포트 충돌
- **API 서버 (8000)**: 다른 프로세스 확인 후 종료
- **프론트엔드 (5173)**: `vite.config.js`에서 포트 변경

### CORS 오류
- API 서버가 실행 중인지 확인
- 프론트엔드 `vite.config.js`의 proxy 설정 확인

### 브라우저 콘솔 오류
- **F12** 키를 눌러 개발자 도구 열기
- **Console** 탭에서 오류 메시지 확인
- 네트워크 탭에서 API 요청 상태 확인
