## 🚀 최종 테스트

### 1. 프론트엔드 접속
```
http://localhost:5173
```

### 2. 테스트 시나리오

#### 시나리오 1: 텍스트 요청
1. 텍스트 입력: "빨간색 원피스 추천해줘"
2. "요청 전송" 클릭
3. 상품 추천 결과 확인

#### 시나리오 2: 이미지 업로드
1. 의류 이미지 업로드 (드래그 앤 드롭 또는 클릭)
2. "요청 전송" 클릭
3. **ChatGarment 서비스가 실제로 처리합니다!**
4. 분석 결과 및 3D 생성 결과 확인

#### 시나리오 3: 이미지 + 텍스트
1. 의류 이미지 업로드
2. 텍스트 입력: "이 옷을 입혀줘"
3. "요청 전송" 클릭
4. 전체 파이프라인 실행 결과 확인

## 📊 예상 결과

- **텍스트 요청**: 상품 추천 목록 (2-3초)
- **이미지 요청**: 이미지 분석 + 3D 생성 (10-30초)

## 🔧 문제 해결

### 서비스가 응답하지 않는 경우

1. **WSL 서비스 확인**:
   ```powershell
   wsl -d Ubuntu -- bash -c "curl http://localhost:9000/health"
   ```

2. **서비스 재시작**:
   ```powershell
   wsl -d Ubuntu -- bash -c "cd ~/ChatGarment/chatgarment_service && python3 main.py"
   ```

3. **API 서버 재시작**:
   ```powershell
   .\restart_api_server.ps1
   ```
